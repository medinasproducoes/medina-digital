Pixels de Remarketing

Como configurar seus pixels

Abra o arquivo index.html e localize a seção <!-- PIXELS --> no topo. Substitua os valores conforme abaixo:

📘
Meta Pixel (Facebook + Instagram)

Acesse: business.facebook.com → Gerenciador de Eventos → Criar Pixel → Copie o ID numérico

var META_PIXEL_ID = 'COLE_SEU_ID_AQUI';
🔴
Google Tag Manager

Acesse: tagmanager.google.com → Criar conta → Copie o ID (GTM-XXXXXXX)

var GTM_ID = 'GTM-XXXXXXX';
🎵
TikTok Pixel

Acesse: ads.tiktok.com → Ativos → Eventos → Pixel Web → Copie o ID

var TIKTOK_ID = 'COLE_SEU_ID_AQUI';
